# warehouse_stock_allocation_agent
warehouse_stock_allocation_agent
What the system does:

Forecasts demand using Exponential Smoothing

Adds Safety Stock (95% service level, 7-day lead time)

Respects Min & Max store capacity

Applies a Priority score (based on distance, margin, strategic importance, etc.)

Allows a Forecast Adjustment Factor for scenario planning

Generates a structured AI summary of the allocation decision
